1. 대시보드 개요

구글 시트의 판매 원장 데이터를 읽어 브랜드·제품·기간·거래처별 매출을 즉시 분석합니다. SENA 전용 배포와 멀티브랜드(MGD) 배포로 나뉘며, DEPLOY_MODE 한 줄로 전환됩니다.

2. 필터 패널

브랜드 (멀티브랜드 배포 전용)

KLIM · SCHUBERTH · TOURATECH · BARKBUSTERS 등 브랜드를 복수 선택합니다. 미선택 상태에서는 조회 결과가 없습니다. 선택된 브랜드에 속한 SKU만 아래 주카테고리·태그·SKU 목록에 표시됩니다.

기간

단위를 기준으로 선택이 동작합니다. 다른 단위를 선택하면 이전 단위의 선택은 자동으로 초기화됩니다.

주카테고리

라디오 버튼으로 1개만 선택합니다. 선택 시 SKU 드롭다운이 해당 카테고리 소속으로 자동 필터링됩니다.

태그

복수 선택 가능합니다(OR 조건). 주카테고리와 동시에 선택하면 AND 조건으로 교집합만 조회합니다.

예: 주카테고리 'Helmet' + 태그 'Krios Pro' → Krios Pro 헬멧만. 태그 'ADV' + 태그 'Summer' → ADV 또는 Summer 태그 SKU 전체.

SKU / 거래처 / 거래유형

SKU는 검색창으로 빠르게 찾을 수 있으며 복수 선택됩니다. 거래처도 복수 선택, 거래유형은 체크박스로 선택합니다.

4. 차트

월별 추이 차트

탭으로 매출액과 판매수량 기준을 전환할 수 있습니다. 라벨은 상세 집계 테이블의 타임컬럼과 동일합니다 (2504, 25Q1 등). 연도 경계에 파란 점선 구분선이 표시됩니다.

상위 5개 SKU의 판매수량 추이가 컬러 라인으로 오버레이됩니다.

비중 도넛 차트

상세 집계의 현재 탭 기준으로 그려집니다. 시리즈뷰이면 시리즈별, 거래처뷰이면 거래처별 비중. 상위 6개 + 기타로 구성됩니다.

5. 상세 집계 테이블

뷰 구성

[시리즈 | SKU] 토글로 같은 매출을 다른 해상도로 봅니다. 시리즈는 색·사이즈 변형을 묶은 모델 단위, SKU는 개별 품번입니다. 그 옆 탭으로 분석축을 전환합니다.

타임컬럼

선택 연도·기간에 따라 자동으로 단위가 결정됩니다. 시리즈·그룹 행(부모)과 아코디언 행(자식) 모두 동일한 타임컬럼 기준으로 집계되어 세로 정렬이 일치합니다.

연도별 이중 표시 (2개년 이상 조회 시)

그룹 행(시리즈/거래처 등 부모 행)과 합계 행에서 판매수량·매출액·평균단가 셀이 연도별로 분리 표시됩니다.

예: 2025+2026 조회 시 → 판매수량 셀에 ● 681 (25년) / ○ 231 (26년) 형태로 표시. SKU 단독 행과 아코디언 자식 행은 단일 표시 유지.

아코디언

시리즈·거래처·월·거래유형 뷰에서 행을 클릭하면 하위 SKU들이 펼쳐집니다. 자식 행의 합계가 부모 행의 숫자입니다.

시리즈뷰의 'SKU수·대표규격명': 하위 SKU들의 공통 앞부분을 추출한 규격명입니다. 예: Krios Pro 시리즈 → "Krios Pro Helmet ECE/DOT" (색·사이즈 부분 제거)

정렬 및 내보내기

컬럼 헤더를 클릭하면 정렬됩니다. 월별 탭은 최신 월 기준 내림차순 고정, 거래유형 탭은 세금계산서 → 소매 → 카드 → 기타 순 고정.

CSV 내보내기: 연도별 이중 표시가 활성화된 경우 2025_판매수량, 2026_판매수량 형태로 연도별 컬럼이 분리 출력됩니다.

6. 발주 예측

과거 판매 데이터 기반으로 발주 권장 수량을 계산합니다. 발주예측 보기 버튼으로 활성화됩니다. 3개월 이상 판매 기록이 없는 SKU는 자동 제외됩니다(단종 또는 판매 저조로 판단).

예측 알고리즘: 트렌드 보정 가중 이동평균

Step 1. 가중 이동평균 (WMA)
WMA = 최근월×W₁ + 2개월전×W₂ + … (최근일수록 높은 가중치)
기간가중치 (최근→과거)
3개월50% · 30% · 20%
6개월25% · 20% · 18% · 15% · 12% · 10%
12개월18% · 14% · 12% · 10% · 9% · 8% · 7% · 6% · 5% · 5% · 4% · 2%
Step 2~4. 트렌드율 → 예측값 → 발주권장수량
트렌드율(%) = 선형회귀 기울기 ÷ WMA × 100 (범위: -50% ~ +50%)
예측값 = WMA × (1 + 트렌드율)
발주권장수량 = 예측값 × 1.2 (안전재고 20% 가산)

컬럼 설명

컬럼설명
월판매(3개월)최근 3개월 WMA + 증감 추세율(↑↓)
월판매(6개월)최근 6개월 WMA + 추세율
월판매(12개월)최근 12개월 WMA + 추세율
1개월 예측신뢰도 높은 기간 기준 트렌드 반영 예측량
3개월 예측1개월 예측 × 3
발주권장수량예측값 × 1.2. 현재 재고를 차감 후 실발주량 결정 권장
판매라이프사이클판매 기록이 있는 개월 수. 짧을수록 신규/단종 가능성
데이터 기간 3개월 미만(⚠ 표시) 제품은 예측 신뢰도가 낮습니다. 참고용으로만 활용하세요.

8. 활용 팁

다년간 추이 분석

2~3개년 선택 + 적용 → 조회. 타임컬럼이 월별(2개년) 또는 분기별(3~4개년)로 자동 전환됩니다. 요약 카드와 합계 행에 연도별 판매수량·매출액·평균단가가 나란히 표시됩니다.

특정 계절 비교

2023+2024 선택 + 4Q 기간 선택 → 조회. 두 해의 10·11·12월 데이터만 타임컬럼으로 표시되어 겨울 성수기 비교가 가능합니다.

컬렉션 전체 매출 조회 (KLIM)

태그에서 컬렉션명(Dakar, Krios Pro, Mojave 등) 선택 → 조회. 자켓·팬츠·글러브 등 카테고리에 관계없이 해당 컬렉션 전체 매출이 집계됩니다.

시리즈 내 SKU 비중 확인

시리즈뷰에서 행 클릭 → 하위 SKU 아코디언 펼침. 각 SKU의 매출액과 그룹 내 비중이 표시됩니다.

반품 많은 제품 찾기

SKU 또는 시리즈뷰로 조회 → 반품율 컬럼 클릭(내림차순) → 빨간 뱃지 제품 집중 관리.

발주 계획 수립

  1. 최근 2~3년 연도 선택 + 적용
  2. 주카테고리로 발주 대상 제품군 선택
  3. 조회 후 발주예측 보기 활성화
  4. 판매라이프사이클 긴 제품 = 신뢰도 높음
  5. 발주권장수량에서 현재 재고 차감 후 실발주
