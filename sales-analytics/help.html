1. 대시보드 개요

미지세계 매출 대시보드는 구글 시트의 판매 원장 데이터를 실시간으로 읽어 분석합니다. 필터 조건을 자유롭게 조합하여 원하는 기간·제품·거래처의 매출을 즉시 확인할 수 있습니다.

2. 필터 패널 사용법

📅 기간

월 버튼(1~12월), 분기(1Q~4Q), 상/하반기, 전체 중 선택하거나 직접 날짜 범위를 입력합니다. 월 버튼과 날짜 범위는 동시에 사용할 수 없습니다.

📅 연도

복수 선택 가능합니다. 여러 연도를 선택하면 합산하여 조회합니다. 전체 연도를 조회하면 시간이 오래 걸릴 수 있습니다.

연도를 선택하지 않으면 전체 연도를 조회합니다 — 수만 행 처리로 수십 초 소요될 수 있습니다.

🏷️ 주카테고리

라디오 버튼으로 1개만 선택합니다. 선택 시 SKU 드롭다운이 해당 카테고리 소속 SKU로 자동 필터링됩니다. 도넛 차트도 품목별 비중으로 전환됩니다.

🔖 태그

복수 선택 가능합니다. 선택한 태그 중 하나라도 포함된 SKU(OR 조건)를 조회합니다. 주카테고리와 태그를 동시에 선택하면 AND 조건으로 교집합 조회합니다.

예: 주카테고리 '자전거' + 태그 '아웃도어' → 자전거 카테고리이면서 아웃도어 태그인 SKU만 조회

📦 SKU

복수 선택 가능합니다. 검색 창으로 특정 SKU를 빠르게 찾을 수 있습니다. 주카테고리 선택 시 목록이 해당 카테고리로 자동 필터됩니다.

🏪 거래처 / 🔖 거래유형

거래처는 복수 선택, 거래유형(소매/세금계산서/카드 등)은 체크박스로 선택합니다.

4. 차트 설명

📈 월별 매출 추이

조회 기간의 월별 매출액(공급가액)을 막대 그래프로 표시합니다.

전년비교 ON: 동일 월의 전년도 매출액을 점선으로 오버레이합니다. 연도를 먼저 선택해야 활성화됩니다.

전년비교는 선택된 연도에서 1년을 뺀 연도의 데이터를 추가로 조회합니다 — 부하가 증가할 수 있습니다.

🥧 비중 도넛 차트

탭으로 매출액 비중과 판매수량 비중을 전환할 수 있습니다.

5. 상세 집계 테이블

뷰 전환 — 묶음 해상도와 분석축

상세 집계는 두 종류의 전환을 제공합니다.

SKU ⇄ 시리즈 토글 (파란 세그먼트): 같은 매출을 보는 해상도를 바꿉니다. SKU는 개별 품번(색·사이즈까지), 시리즈는 그 묶음(Krios Pro, 50S 등)입니다. 시리즈뷰에서 행을 클릭하면 하위 SKU들이 아코디온으로 펼쳐집니다.

분석축 탭 (거래처·월별·거래유형): 같은 매출을 다른 관점으로 봅니다.

6. 발주 예측

과거 판매 데이터를 기반으로 향후 발주에 필요한 예측 수량을 계산합니다. 📦 발주예측 보기 버튼으로 활성화됩니다.

예측 알고리즘: 트렌드 보정 가중 이동평균

Step 1. 가중 이동평균 (Weighted Moving Average)
WMA = (최근월×W₁) + (2개월전×W₂) + (3개월전×W₃) + ...
기간가중치 (최근→과거)
3개월50% · 30% · 20%
6개월25% · 20% · 18% · 15% · 12% · 10%
12개월18% · 14% · 12% · 10% · 9% · 8% · 7% · 6% · 5% · 5% · 4% · 2%
Step 2. 트렌드(증감 추세) 계산 — 선형 회귀
기울기(slope) = 기간 내 월별 판매량의 최소자승법(OLS) 직선 기울기
트렌드율(%) = slope ÷ 가중평균 × 100
(범위 제한: −50% ~ +50%)
트렌드율 예시: 3개월 평균이 100개인데 매월 8개씩 증가 추세 → ↑+8%. 즉 "현재 월 판매량이 평균 대비 8% 속도로 증가 중"을 의미합니다.
Step 3. 트렌드 보정 예측값
예측값 = WMA × (1 + 트렌드율)
Step 4. 발주권장수량 (안전재고 포함)
발주권장수량 = 예측값 × 1.2 (안전재고 20% 가산)

8. 활용 팁

빠른 브랜드별 분석

주카테고리로 제품군을 먼저 선택한 후 조회하면 해당 카테고리의 SKU별 매출, 반품율, 비중을 한눈에 볼 수 있습니다.

전년도 성장률 확인

현재 연도를 선택 후 조회 → 전년비교 ON → 막대차트에서 전년도 점선과 현재 매출 막대를 비교합니다.

반품 많은 제품 찾기

SKU 그룹으로 조회 후 반품율 컬럼 클릭(내림차순 정렬) → 빨간 뱃지 SKU 집중 관리.

발주 계획 수립

  1. 연도: 전체 또는 최근 2~3년 선택
  2. 주카테고리로 발주 대상 제품군 선택
  3. 조회 후 발주예측 보기 활성화
  4. 판매라이프사이클 긴 제품 = 안정적 데이터 → 신뢰도 높음
  5. 발주권장수량에서 현재 재고 차감 후 실발주

CSV 내보내기

상세 집계 테이블과 발주 예측 모두 CSV 다운로드를 지원합니다. 엑셀에서 추가 분석이 가능합니다.
